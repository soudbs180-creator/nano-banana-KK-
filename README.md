# KK Studio v1.1.7 - AI 智能图像创作工坊

> 基于 Google Gemini API 的无限画布图像生成工具，集成本地与云端同步的高级创作平台。

![Version](https://img.shields.io/badge/version-1.1.7-blue.svg)

## 🌟 核心功能 (Core Features)

### 🎨 无限画布引擎 (Infinite Canvas)
- **节点化创作**：基于节点的非线性创作流，支持分支与迭代。
- **自由交互**：支持无限平移、缩放、多选拖拽。
- **智能排版**：一键自动整理（Auto Arrange），智能归纳孤儿节点与关联节点，保持画布整洁。

### 🤖 AI 智能生成 (AI Generation)
- **多模型支持**：集成 **Gemini 3 Pro** (顶级推理)、**Gemini 3 Flash** (高速生成) 及 **Imagen 3/4** 等多款模型。
- **参数精调**：支持自定义宽高比（16:9, 9:16, 1:1, 4:3 等）与分辨率（1K/2K/4K）。
- **参考图生成**：支持上传参考图（Reference Image）进行风格或构图控制。
- **原地重绘**：节点原地更新，保留创作上下文。

### 💬 智能聊天助手 (AI Assistant)
- **上下文对话**：内置悬浮聊天助手，支持上下文连续对话。
- **模型切换**：即时切换不同对话模型（如 Gemini Flash Lite, Pro 等）以适应不同任务。
- **小巧交互**：悬浮球设计，支持自动展开、拖拽移动。
- **成本估算**：聊天 Token 消耗实时计入成本统计。

### 🖼️ 高级图像管理 (Image Management)
- **沉浸式查看器**：
  - **滚轮缩放**：精准控制图片细节。
  - **按住拖拽**：在放大状态下自由移动视角。
  - **双击关闭**：快速退出查看模式。
- **原图下载**：支持下载无损原图，自动处理文件名。
- **批量管理**：支持单张删除或通过画布清理无效节点。

### ⚙️ 系统与设置 (System & Settings)
- **成本追踪**：实时估算 API 调用成本（USD），支持 Token 与固定费率混合计价。
- **API 负载均衡**：支持多 Key 轮询（Key Rotation），自动处理 API 限流问题。
- **灵活存储**：支持 **本地文件系统**（直接保存到硬盘）或 **浏览器存储**（IndexedDB）两种模式。
- **One API 面板**：统一管理多个 API Key，支持一键验证与状态监控。

## 🚀 快速启动

```bash
npm run dev
```

1. **登录**：首次使用需注册/登录账号（本地离线账户）。
2. **配置**：点击左下角头像 -> One API 面板 -> 添加 Google Gemini API Key。
3. **创作**：在底部输入框输入提示词，开始您的创作之旅。

## 🛠️ 操作快捷键

| 功能 | 操作方式 |
|------|----------|
| **画布平移** | 按住空格 + 拖拽 / 鼠标中键拖拽 |
| **画布缩放** | 鼠标滚轮 |
| **查看原图** | 双击图片节点 |
| **关闭原图** | 双击原图 / 点击背景 |
| **移动原图** | 查看模式下按住左键拖拽 |
| **删除节点** | 选中后按 Delete 键 / 点击垃圾桶图标 |

## 📦 技术与依赖

- **核心架构**：React 19, TypeScript, Vite

- **后端服务**：Supabase (Auth/DB), Google Gemini API
- **部署支持**：Netlify, Vercel
