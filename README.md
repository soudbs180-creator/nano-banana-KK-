# KK Studio v1.1.1 - AI 图像创作工作室

> 基于 Google Gemini API 的无限画布图像生成工具，集成本地与云端同步的高级创作平台。

![Version](https://img.shields.io/badge/version-1.1.1-blue.svg)

## 🚀 快速启动

**双击 `KK Studio.exe`** 启动应用。
*   **首次使用**：需注册账号并登录。
*   **权限管理**：系统强制要求在所有环境（包括本地和线上）进行身份验证，确保数据安全。

## ✨ v1.1.1 新特性

### 🔐 账号与云同步
- **用户系统**：支持邮箱注册、登录、密码重置及修改。
- **云端 Key 同步**：您的 API Key 现已安全存储于云端（Supabase），并在多设备间自动同步。
- **本地零残留**：退出登录或切换账号时，自动清除本地 API Key 缓存，防止隐私泄露。
- **Apple 风格设置**：全新的个人中心 UI，提供丝滑的操作体验。

### 🎨 图像生成
- **模型支持**：Gemini Pro (高清) 与 Flash (极速)。
- **智能排版**：生成结果自动在画布上智能排列，单图自动居中。
- **多比例支持**：1:1、16:9、9:16、4:3 等常用比例。
- **参考图引导**：支持上传参考图进行风格化生成。

### 💬 移动端 AI 助手
- **悬浮球入口**：移动端底部新增智能对话悬浮球。
- **全屏体验**：类似 iOS 的对话界面，支持上下文连续对话。

## 🛠️ 操作指南

### 1. 配置 API Key
1. 点击左侧栏底部的**个人头像**。
2. 选择 **"API Key 设置"**。
3. 输入您的 Google Gemini API Key（支持多个 Key 轮询）。
4. 系统会自动验证并保存至云端。

### 2. 图像创作
- **输入提示词**：在底部输入框描述您想要的画面。
- **调整参数**：点击输入框左侧设置图标，选择比例、模型和生成数量。
- **开始生成**：按 Enter 发送。

### 3. 无限画布交互
- **平移**：按住鼠标左键或中键拖拽空白处。
- **缩放**：滚动鼠标滚轮。
- **卡片操作**：
    - **拖拽**：移动卡片位置。
    - **连线**：从卡片底部的圆点拖拽连线到其他卡片。
    - **下载**：点击卡片底部的下载图标。

## 📦 技术栈

- **前端**：React 19 + TypeScript + Vite
- **样式**：Tailwind CSS + Vercel Geist Font
- **后端服务**：Supabase Auth & Database
- **AI**：Google Gemini Pro Vision

## ⚠️ 注意事项

*   本应用需配合有效的 **Google Gemini API Key** 使用。
*   请勿将 Key 泄露给他人。
*   如遇到 "Backend function not found" 错误，请确保您已配置 API Key，系统将优先尝试直连生成。
