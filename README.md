# KK Studio v1.1.4 - AI 图像创作工作室

> 基于 Google Gemini API 的无限画布图像生成工具，集成本地与云端同步的高级创作平台。

![Version](https://img.shields.io/badge/version-1.1.4-blue.svg)

## 🚀 快速启动

```bash
npm run dev
```

- **首次使用**：注册账号并登录
- **存储选择**：登录后选择原图保存位置（本地文件夹/浏览器）
- **API 配置**：添加 Google Gemini API Key

---

## ✨ v1.1.4 新特性

### 💾 存储选择系统
- **登录后弹窗**：选择原图保存方式
  - **本地文件夹**：原图直接保存到本地指定文件夹
  - **浏览器存储**：原图保存在 IndexedDB
- **云端缩略图**：缩略图始终同步到云端，任何设备可预览
- **个人中心设置**：随时更改存储位置

### 🔔 全局通知系统
- 屏幕顶部居中显示
- 错误信息支持一键复制（AI 调试友好）
- 四种类型：成功/错误/警告/信息

### 🎨 画布自动排版
- 一键整理卡片布局
- 提示词在上，生成图片在下
- 4 列紧凑网格，无重叠

## 🛠️ 操作指南

### 1. 配置 API Key
1. 点击左侧栏底部的 **个人头像**
2. 选择 **"One API 面板"**
3. 添加 Google Gemini API Key（支持多 Key 轮询）

### 2. 图像创作
- **输入提示词**：底部输入框描述画面
- **调整参数**：比例、尺寸、模型、生成数量
- **开始生成**：按 Enter 或点击发送

### 3. 无限画布交互
| 操作 | 方式 |
|------|------|
| 平移 | 鼠标拖拽空白处 |
| 缩放 | 滚轮 |
| 移动卡片 | 拖拽卡片 |
| 自动排版 | 左下角"整理排版"按钮 |
| 下载原图 | 点击卡片底部下载按钮 |
| 放大查看 | 双击图片 |

### 4. 存储位置设置
1. 点击 **个人头像** → **个人中心**
2. 选择 **"存储位置"**
3. 切换至本地文件夹或浏览器存储

- **样式**：Tailwind CSS
- **后端**：Supabase Auth & Database
- **AI**：Google Gemini API

## ⚠️ 注意事项

- 需配合有效的 **Google Gemini API Key**
- 本地文件夹模式需 Chrome/Edge 86+ 浏览器
- 切换浏览器后原图需重新下载（缩略图始终可见）
