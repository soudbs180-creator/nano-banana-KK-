# KK Studio v1.1.5 - AI 图像创作工作室

> 基于 Google Gemini API 的无限画布图像生成工具，集成本地与云端同步的高级创作平台。

![Version](https://img.shields.io/badge/version-1.1.5-blue.svg)

## 🚀 快速启动

```bash
npm run dev
```

- **首次使用**：注册账号并登录
- **存储选择**：登录后选择原图保存位置（本地文件夹/浏览器）
- **API 配置**：添加 Google Gemini API Key

## ✨ v1.1.5 更新日志

### 🔄 智能重绘与 API 容错
- **原地重绘 (In-Place Regeneration)**：重新生成时不再删除原卡片，直接在当前位置更新，保持画布布局稳定。
- **自动容错 (Key Rotation)**：API 调用失败时（如 429 限流），自动在后台无缝切换备用 Key（最多尝试 50 次），大幅提升成功率。

### 📐 布局优化
- **智能网格排版**：自动整理（Auto Arrange）现在支持每行 **10 个项目** 后自动换行，避免画布无限横向延伸。
- **孤儿图像整理**：未关联提示词的图像会自动排列在画布末尾，不再重叠混乱。
- **连线修复**：修复了“继续创作”时连接线无法正确指向后续提示词的问题。

### 💰 精准成本估算
- **多模型计费**：支持不同模型的精确计费
  - **Imagen**：固定费率 ($0.02 - $0.06/张)
  - **Gemini 3 Pro**：Token 计费 ($2.00/1M 输入, $120.00/1M 输出)
  - **Gemini 2.5 Flash**：标准计费 ($0.10/输入, $0.40/输出)
- **Token 追踪**：记录每次生成的输入 Prompt 长度和参考图 Token 消耗。

## 🛠️ 操作指南

### 1. 配置 API Key
1. 点击左侧栏底部的 **个人头像**
2. 选择 **"One API 面板"**
3. 添加 Google Gemini API Key（支持多 Key 轮询）

### 2. 图像创作
- **输入提示词**：底部输入框描述画面
- **调整参数**：比例、尺寸、模型、生成数量
- **开始生成**：按 Enter 或点击发送

### 3. 无限画布交互
| 操作 | 方式 |
|------|------|
| 平移 | 鼠标拖拽空白处 |
| 缩放 | 滚轮 |
| 移动卡片 | 拖拽卡片 |
| 自动排版 | 左下角"整理排版"按钮 |
| 下载原图 | 点击卡片底部下载按钮 |
| 放大查看 | 双击图片 |

### 4. 存储位置设置
1. 点击 **个人头像** → **个人中心**
2. 选择 **"存储位置"**
3. 切换至本地文件夹或浏览器存储
