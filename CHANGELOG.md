# 更新日志

## v1.1.9 (2026-01-21)

### ✨ 新功能
- **第三方 API 代理支持**：支持 gemini-balance-lite 等中转服务，可在"添加密钥"时开启代理模式并输入 Base URL
- **动态 API 路由**：自动根据 Key 配置选择官方 API 或代理端点

### 🔧 改进
- **计费精度更新**：Gemini 3 Pro / 2.5 Flash 图片生成价格已按 Google 官方文档校准
- **超时时长调整**：图片生成 API 超时从 45 秒延长至 4 分钟，避免复杂任务中断

### 🏗️ 技术改进
- 新增 `apiConfig.ts` 统一管理 API 配置
- `keyManager.ts` 支持 `baseUrl` / `authMethod` / `headerName` 字段
- `geminiService.ts` / `chatService.ts` 使用 `buildApiUrl()` 动态构建请求

---

## v1.1.6 (2026-01-19)

### 🐛 修复
- **连接线定位**：修复了提示词卡片连接线端点偏移的问题，现在基于卡片实际高度动态定位，无论提示词长短均能精准连接。
- **构建错误**：修复了生产环境构建时丢失 `getModelDisplayName` 的问题。

## v1.1.5 (2026-01-19)

### ✨ 新功能
- **原地重绘**：点击"Regenerate"直接在当前卡片重试，不再闪烁或移动位置。
- **API 自动容错**：生成失败时自动重试并切换 API Key（最大 50 次）。
- **智能布局 (Wrap)**："Auto Arrange" 排版现在每 10 个项目自动换行。
- **孤儿图像整理**：未关联图像整齐排列在画布末尾。
- **精准计费**：集成 Imagen/GeminiPro/Flash 的精确计费模型。

### 🔧 改进
- **占位符高度**：生成时的 Loading 占位符高度现在与最终卡片完全一致 (含 Footer)，消除布局跳动。

---

## v1.1.4 (2026-01-19)

### ✨ 新功能
- **存储选择弹窗**：登录后选择原图保存位置（本地/浏览器）

- **画布自动排版**：一键整理卡片为 4 列网格（提示词在上，图片在下）

### 🔧 改进

- **错误提示增强**：Supabase 错误显示详细代码和提示

### 🐛 修复
- 删除无用 `ImageCard.tsx`（4 个 TypeScript 错误）
- 移除登录框默认占位文本
