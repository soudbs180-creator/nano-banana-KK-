# 更新日志

## v1.1.3 (2026-01-16)

### 🐛 Bug 修复
- **个人中心黑屏**: 添加 ErrorBoundary 错误边界，渲染错误时显示详细信息而非空白
- **刷新丢失图片**: 优化 IndexedDB 持久化，参考图片现在也会保存到 IndexedDB
- **整齐排列堆叠**: 增加行间距到 800px，防止纵向卡片重叠
- **不同尺寸卡片堆叠**: 添加 footer 高度计算 (60px)，修复布局溢出
- **拖动卡片复制文字**: 添加 `select-none` 类，拖动时不再选中文字
- **生成时卡片位置**: 拖动生成中的卡片后，生成的图片会出现在新位置

### ✨ 功能增强
- **生成时间记录**: 每张图片现在显示生成耗时 (如 "2.5s")
- **API 并发调配**: 多张图片生成时自动轮询使用不同 API Key
- **API 配额显示**: 进度条显示配额百分比，倒计时显示重置时间
- **手机端优化**: 验证移动端功能一致性，UI 自适应

### 🔧 技术改进
- 重构图片位置计算逻辑，使用实时卡片位置
- localStorage 保存防抖 (500ms)，减少写入频率
- 添加 localStorage 配额检测和降级处理
