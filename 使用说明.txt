KK Studio - AI 图像生成工作站
===============================

快速开始
--------
双击 "KK Studio.exe" 启动应用程序。

浏览器将自动打开 http://localhost:3000


使用方法
--------
1. 启动
   - 双击：KK Studio.exe
   - 首次运行会自动安装依赖（需要联网）
   - 服务就绪后浏览器自动打开

2. 停止  
   - 双击：stop.bat
   - 这将关闭后台服务

3. API 设置
   - 点击右上角头像图标
   - 输入您的 Google Gemini API 密钥
   - 密钥会本地保存，无需重复输入


常见问题
--------
问：启动后没反应？
答：请确保已安装 Node.js（推荐 v18 或更高版本）。
    下载地址：https://nodejs.org/

问：提示 "找不到 startup.bat"？
答：所有文件必须一起解压。不要单独移动 KK Studio.exe。

问：生成图片失败？
答：请检查 API 密钥是否有效以及网络连接是否正常。


文件说明
--------
KK Studio.exe  - 主启动器（双击启动）
startup.bat    - 启动脚本（由 exe 调用）
stop.bat       - 停止服务脚本
Build EXE.ps1  - 重新生成 exe（如需要）
使用说明.txt    - 本文件
